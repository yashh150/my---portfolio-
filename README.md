## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/yashh150/Portfolio.git
```

2. Change the working directory

```bash
cd Portfolio
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm start
```

## 🚀 Features

- Fully Responsive
- Multi-Page Layout
- Contact Form
- Blog Section
- Portfolio Section
- Resume Section
- Dynamic Project Section
- Search Bar
